# Exercise 4 - Creating and Configuring a Maven Project

## Objective
This project demonstrates how to create and configure a Maven project with the required Spring Framework dependencies and Maven Compiler Plugin.

## Technologies Used
- Java 8
- Maven
- Spring Context
- Spring AOP
- Spring Web MVC

## Project Structure

```
Exercise4_MavenConfiguration
├── pom.xml
├── src
├── output
└── README.md
```

## Dependencies
- Spring Context
- Spring AOP
- Spring Web MVC

## Maven Plugin
- Maven Compiler Plugin (Java 1.8)

## Output
The application confirms that the Maven project has been configured successfully.
